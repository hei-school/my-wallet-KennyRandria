# Java WALLET App

This Java application represents a portfolio system with basic functionalities.
Users can deposit, withdraw, check their balance, and view transaction history.
Additionally, the application collects user information,
including CIN, card number, Visa, driving license, and photo ID.

## How to Run

1. Make sure you have Java installed on your system.
2. Compile and run the Wallet.java file.

```bash
javac Wallet.java
java Wallet
