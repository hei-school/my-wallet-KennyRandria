
# PHP Wallet App

This PHP application represents a wallet system with basic functionalities.
Users can deposit, withdraw, check their balance,
 and view transaction history. 
Additionally, the application collects user information,
including CIN, card number, Visa, driving license, and photo ID.

## How to Run

1. Ensure you have a PHP-enabled web server installed.
2. Copy the PHP files to your web server directory.
3. Access the PHP files through a web browser.
4. Follow the on-screen instructions for authentication and wallet operations.

## Features

- Deposit money into the wallet.
- Withdraw money from the wallet.
- Check current balance.
- View transaction history.
- User authentication.
- Collect and display user information.
