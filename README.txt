# Python Wallet App

This Python application represents a wallet system with basic functionalities.
Users can deposit, withdraw, check their balance, and view transaction history.
Additionally, the application collects user information,
 including CIN, card number, Visa, driving license,and photo ID.

## How to Run

1. Ensure you have Python installed on your system.
2. Run the Python script.

```bash
python Wallet.py
